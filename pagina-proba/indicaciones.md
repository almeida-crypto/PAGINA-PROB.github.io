1. Objetivo

Desarrollar una aplicación web interactiva que implemente conceptos fundamentales de 
probabilidad y estadística.
2. Lenguaje de programación
El lenguaje de programación es LIBRE. Cada alumno puede utilizar las tecnologías que 
prefiera (HTML, CSS, JavaScript, frameworks, librerías, etc.), siempre y cuando el resultado 
final sea una aplicación web funcional publicada en GitHub Pages.
La evaluación se centrará en el funcionamiento correcto de los cálculos, la implementación 
de los conceptos estadísticos y la calidad del proyecto, no en el lenguaje específico utilizado.
3. Requisitos del proyecto
• Ingreso manual de datos numéricos (mínimo 20 datos).
• Generación de datos aleatorios. (mínimo 20 datos).
• Cálculo de media, mediana, moda, mínimo, máximo y rango en base a los datos.
• Tabla de frecuencias (fi, fr, Fi, Fr).
• Histograma.
• Polígono de frecuencias.
• Ojiva.
• Diagrama de Pareto.
• Operaciones con conjuntos.
• Probabilidad.
• Regla multiplicativa y diagrama de árbol.
• Permutaciones y combinaciones.
4. Publicación en GitHub Pages
1. Crear un repositorio público en GitHub.
2. Subir los archivos del proyecto (index.html en la raíz).
3. Activar GitHub Pages desde la configuración del repositorio.
4. Compartir únicamente la URL pública generada
