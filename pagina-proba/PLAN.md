# Plan de Desarrollo: Probabilidad y Estadística Web

## 1. Estructura y UI (Interfaz de Usuario)
- [ ] `index.html`: Estructura base con navegación por pestañas (Estadística, Gráficos, Probabilidad, Conjuntos).
- [ ] `style.css`: Diseño moderno (Glassmorphism o Material Design), responsivo y profesional.
- [ ] Integración de **Chart.js** vía CDN para visualizaciones.

## 2. Módulo de Datos (Entrada)
- [ ] Implementar área de texto para ingreso manual de datos (mínimo 20).
- [ ] Botón para generación de datos aleatorios (mínimo 20).
- [ ] Limpieza y validación de datos (solo números).

## 3. Estadística Descriptiva
- [ ] Cálculo de: Media, Mediana, Moda, Mínimo, Máximo y Rango.
- [ ] Generación automática de **Tabla de Frecuencias**:
  - `fi`: Frecuencia absoluta.
  - `fr`: Frecuencia relativa.
  - `Fi`: Frecuencia absoluta acumulada.
  - `Fr`: Frecuencia relativa acumulada.

## 4. Visualización (Gráficos)
- [ ] **Histograma**: Distribución de frecuencias.
- [ ] **Polígono de Frecuencias**: Línea sobre el histograma.
- [ ] **Ojiva**: Gráfico de frecuencias acumuladas.
- [ ] **Diagrama de Pareto**: Frecuencia absoluta + porcentaje acumulado.

## 5. Operaciones con Conjuntos
- [ ] Interfaz para ingresar Conjunto A y B.
- [ ] Operaciones: Unión, Intersección, Diferencia, Complemento.
- [ ] Representación visual (Diagrama de Venn simple).

## 6. Probabilidad y Combinatoria
- [ ] Calculadora de **Permutaciones** y **Combinaciones**.
- [ ] Calculadora de Probabilidad básica (Casos favorables / Casos totales).
- [ ] **Diagrama de Árbol**: Generador visual para la regla multiplicativa.

## 7. Publicación
- [ ] Preparación final para GitHub Pages.
